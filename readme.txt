== College Theme  ==
College is a responsive WordPress theme for university, school and teacher websites. Some of the features that the College theme includes are multiple columns, homepage, customizer options, sliders,  recommended shortcodes, responsive layout (built on top of Twitter Bootstrap boilerplate), full width pages, pages with sidebars and more.
== Description ==

College is a responsive WordPress theme for university, school and teacher websites. Some of the features that the College theme includes are multiple columns, homepage, customizer options, sliders,  recommended shortcodes, responsive layout (built on top of Twitter Bootstrap boilerplate), full width pages, pages with sidebars and more.

= Theme Features Usage =
All available options can be used from Appearance->Customize
Theme supports no more than three(3) levels deep navigation.
